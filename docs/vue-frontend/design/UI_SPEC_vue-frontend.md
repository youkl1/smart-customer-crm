# UI设计规范文档 - 智客通CRM

## 1. 设计系统概述
- **系统名称**：智客通CRM（SmartCustomer CRM）
- **设计理念**：简洁、专业、高效，注重用户体验和操作便捷性
- **设计风格**：现代商务风格，使用简洁的线条和清晰的层次结构
- **适用端**：移动端小程序（固定宽度375px）和WEB端后台管理

## 2. 移动端设计规范

### 2.1 布局规范
- **整体布局**：固定宽度375px，水平居中，高度100vh自适应
- **导航结构**：底部导航栏（客户、商机、任务、我的）+ 顶部标题栏
- **内容区域**：垂直滚动，支持下拉刷新和上拉加载
- **弹窗**：居中显示，覆盖背景

### 2.2 尺寸规范
- **屏幕宽度**：375px（固定）
- **底部导航栏**：高度50px
- **顶部标题栏**：高度44px
- **按钮高度**：40px
- **输入框高度**：40px
- **列表项高度**：48px
- **间距**：
  - 组件间距：16px
  - 内边距：12px
  - 外边距：16px

### 2.3 色值规范
- **主色调**：#1890FF（蓝色）
- **辅助色**：
  - 成功：#52C41A
  - 警告：#FAAD14
  - 错误：#F5222D
  - 信息：#1890FF
- **中性色**：
  - 背景色：#F5F5F5
  - 卡片背景：#FFFFFF
  - 文本颜色：
    - 主文本：#333333
    - 次文本：#666666
    - 弱文本：#999999
  - 边框颜色：#E8E8E8

### 2.4 字体规范
- **字体**：PingFang SC（中文）、Helvetica（英文）
- **字号**：
  - 标题：18px
  - 正文：14px
  - 说明文本：12px
  - 按钮文本：14px
- **字重**：
  - 标题：500
  - 正文：400
  - 按钮文本：500

### 2.5 组件规范
- **按钮**：
  - 主要按钮：蓝色背景，白色文字
  - 次要按钮：白色背景，蓝色边框和文字
  - 文字按钮：无背景，蓝色文字
- **输入框**：
  - 高度40px，圆角4px
  - 边框颜色：#E8E8E8
  - 聚焦时边框颜色：#1890FF
- **列表**：
  - 列表项高度48px
  - 底部有1px分割线
  - 支持左滑操作
- **卡片**：
  - 圆角8px
  - 阴影：0 2px 8px rgba(0, 0, 0, 0.08)
  - 内边距16px
- **图标**：
  - 使用Vant UI内置图标
  - 图标尺寸：24px

### 2.6 交互规范
- **触摸反馈**：点击按钮时有轻微的缩放效果
- **加载状态**：使用Vant的Loading组件
- **下拉刷新**：使用Vant的PullRefresh组件
- **上拉加载**：使用Vant的List组件
- **弹窗**：使用Vant的Dialog和Popup组件
- **导航切换**：底部导航栏切换时无动画效果

### 2.7 适配规则
- **屏幕高度**：自适应，内容区域垂直滚动
- **字体大小**：使用px单位，确保在不同设备上显示一致
- **图片**：使用响应式图片，确保在不同设备上显示清晰

- **表格操作**：支持排序、筛选、分页
- **弹窗**：使用Element Plus的Dialog组件
- **导航折叠**：左侧导航栏支持点击按钮折叠/展开

### 3.7 响应式适配规则
- **屏幕宽度**：最小宽度1366px，最大宽度自适应
- **导航栏**：在小屏幕设备上自动折叠
- **表格**：在小屏幕设备上显示横向滚动条
- **布局**：在不同屏幕尺寸下保持布局的一致性

## 6. 全场景设计覆盖

### 6.1 正常状态
- 所有功能正常显示，交互流畅
- 数据加载正常，页面响应及时

### 6.2 异常状态
- **网络错误**：显示网络连接失败提示
- **登录失败**：显示登录失败提示
- **权限不足**：显示无权限访问提示
- **操作失败**：显示操作失败提示

### 6.3 空状态
- **列表为空**：显示空状态提示，引导用户进行操作
- **搜索无结果**：显示搜索无结果提示，引导用户调整搜索条件
- **数据加载为空**：显示空状态提示，引导用户添加数据

### 6.4 加载状态
- **页面加载**：显示页面加载动画
- **数据加载**：显示数据加载动画
- **提交操作**：显示提交中提示

### 6.5 禁用状态
- **按钮禁用**：按钮显示禁用状态，无法点击
- **输入框禁用**：输入框显示禁用状态，无法输入
- **链接禁用**：链接显示禁用状态，无法点击

## 8. 与现有设计系统的适配方案

### 8.1 移动端适配
- 使用Vant UI组件库，确保与设计规范一致
- 自定义主题配置，覆盖默认样式，确保与设计规范一致
- 使用Vant的viewport适配方案，确保在不同设备上显示一致

### 8.2 WEB端适配
- 使用Element Plus组件库，确保与设计规范一致
- 自定义主题配置，覆盖默认样式，确保与设计规范一致
- 使用响应式布局，确保在不同屏幕尺寸下显示一致

### 8.3 设计系统维护
- 建立设计组件库，确保组件的一致性
- 定期更新设计规范，适应业务需求的变化
- 确保设计规范与代码实现的一致性
