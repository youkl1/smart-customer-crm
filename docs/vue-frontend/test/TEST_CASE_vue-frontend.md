## 详细测试用例

### 1. 端选择与登录模块

#### 1.1 端选择页面
- **测试用例ID**：TC-001
- **测试场景**：端选择页面访问
- **测试步骤**：
  1. 访问系统首页
  2. 查看端选择页面显示
  3. 点击WEB端按钮
  4. 点击移动端按钮
- **预期结果**：
  - 页面显示系统Logo和名称
  - 显示WEB端和移动端选择按钮
  - 点击WEB端按钮跳转到WEB端登录页
  - 点击移动端按钮跳转到移动端登录页

#### 1.2 登录功能
- **测试用例ID**：TC-002
- **测试场景**：账号密码登录
- **测试步骤**：
  1. 进入登录页面
  2. 输入正确的账号密码（admin/123456）
  3. 点击登录按钮
- **预期结果**：
  - 登录成功
  - 跳转到对应端的首页
  - 登录状态正确存储

#### 1.3 登录失败
- **测试用例ID**：TC-003
- **测试场景**：账号密码错误
- **测试步骤**：
  1. 进入登录页面
  2. 输入错误的账号或密码
  3. 点击登录按钮
- **预期结果**：
  - 登录失败
  - 显示错误提示信息
  - 页面停留在登录页

#### 1.4 路由守卫
- **测试用例ID**：TC-004
- **测试场景**：未登录访问受保护页面
- **测试步骤**：
  1. 清除登录状态
  2. 直接访问受保护页面（如客户列表）
- **预期结果**：
  - 自动跳转到登录页面
  - 登录后才能访问受保护页面

#### 1.5 端类型守卫
- **测试用例ID**：TC-005
- **测试场景**：移动端路由访问WEB端页面
- **测试步骤**：
  1. 登录移动端
  2. 尝试访问WEB端路由
- **预期结果**：
  - 路由守卫拦截
  - 提示或跳转到合适页面
