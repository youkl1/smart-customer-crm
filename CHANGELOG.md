# 项目变更日志

## [v1.0.0] - 2024-01-01

### 新增功能
- 实现了端选择与统一登录模块
- 实现了移动端小程序功能：
  - 客户管理：客户列表、详情、新增、编辑、公海池
  - 联系人管理：联系人列表、新增、编辑
  - 商机管理：商机列表、详情、新增、编辑、阶段推进
  - 任务管理：任务列表、新增、编辑、完成
  - 跟进记录：添加跟进记录
  - 个人中心：个人信息编辑、修改密码
- 实现了WEB端后台管理功能：
  - 产品管理：产品分类管理
  - 回款管理：回款计划、回款记录、新增回款、编辑回款
  - 报表中心：数据概览、销售漏斗、业绩趋势、客户分析
  - 系统设置：组织架构、角色权限、字典管理、系统日志
  - 个人中心：个人信息、修改密码
- 实现了路由管理、状态管理、API请求封装等核心功能
- 提供了完整的模拟数据

### 修复问题
- 修复了路由配置错误，确保所有路由指向正确的组件
- 修复了移动端底部导航栏宽度异常问题
- 修复了新增客户按钮布局超出范围问题
- 修复了标题头重复问题
- 修复了数据绑定问题，确保模拟数据正确显示
- 修复了Vue Router JSON解析错误
- 修复了WEB端菜单空白问题

### 优化改进
- 创建设计系统变量，统一UI样式
- 优化了移动端布局，提升用户体验
- 优化了弹窗提示，使用Vant的showToast组件
- 优化了WEB端左侧导航栏，支持滚动
- 优化了客户管理列表和详情页UI
- 完善了顶部按钮功能，实现了与子组件的通信机制
- 优化了新增客户按钮的布局，确保在不同屏幕尺寸下都能正确显示
